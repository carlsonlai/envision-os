Integration Setup Guide

Step-by-step instructions to connect WhatsApp and Lark to Envision OS

Before you start: Make sure your dev server is running and you have access to your .env file in the project root. After editing the .env, restart the server for changes to take effect.
💬
WhatsApp (via 360dialog)
Send & receive WhatsApp messages from clients inside Envision OS
1

Create a 360dialog account

Go to hub.360dialog.com and sign up. You need an existing WhatsApp Business number to connect — this is the number your clients will message.

2

Get your API key

Inside the 360dialog Hub: go to Developers → API Key → Generate API Key. Copy the key.

# .env
WHATSAPP_360DIALOG_API_KEY=your_api_key_here
3

Register the webhook URL

In 360dialog Hub → Webhooks → Add Webhook, enter your public URL:

https://yourdomain.com/api/whatsapp/webhook

For local development use ngrok: run ngrok http 3000 and use the HTTPS URL it gives you.

4

Restart the server & test

Stop and restart npm run dev. Then send a WhatsApp message to your registered number — it will appear in Sales → WhatsApp inside the app.

What works

Send & receive text messages, AI-drafted replies, attach conversation to a client project, WhatsApp page in Sales module

Limitations

Outbound messaging to new contacts requires an approved WhatsApp message template. Media (images, files) not yet wired up.

🪶
Lark (Feishu)
Auto-create project folders in Lark Drive, send rich card notifications to team channels
1

Create a Lark app

Go to open.larksuite.com/app (or open.feishu.cn/app for the China version) → Create App → Custom App. Give it a name like "Envision OS".

2

Copy App ID & App Secret

From your app's Credentials & Basic Info tab, copy the App ID and App Secret.

LARK_APP_ID=cli_xxxxxxxxxxxxxx
LARK_APP_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
3

Enable permissions

In your app → Permissions & Scopes, add these:

im:message:send_as_bot drive:drive drive:file drive:folder

Then go to Version Management → Create Version → Submit for Release to apply the permissions.

4

Set up the webhook & verification tokens

In your app → Event Subscriptions → Request URL, enter:

https://yourdomain.com/api/lark/webhook

Lark will give you a Verification Token and Encrypt Key on this page. Copy them:

LARK_VERIFY_TOKEN=xxxxxxxxxxxxxxxx
LARK_ENCRYPT_KEY=xxxxxxxxxxxxxxxx

Subscribe to the event: im.message.receive_v1

5

Set up the root Drive folder

In Lark Drive, create a folder called "Envision OS Projects". Open it — the URL contains the folder token: https://…/drive/folder/FolderToken. Copy it.

LARK_ROOT_FOLDER_TOKEN=fldcnXXXXXXXXXX
6

Connect team notification channels

For each team group chat in Lark, right-click → Get Chat Info → copy the Chat ID. Paste into:

LARK_CHANNEL_CREATIVE=oc_xxxxxxxx
LARK_CHANNEL_CS=oc_xxxxxxxx
LARK_CHANNEL_SALES=oc_xxxxxxxx
LARK_CHANNEL_MANAGEMENT=oc_xxxxxxxx

Make sure you've added the bot to each group chat (+ Add Bot → Envision OS).

7

Restart & test

Restart the dev server. Create a new project in Admin → Projects — it will automatically create a structured folder tree in your Lark Drive root folder and send a notification to the CS channel.

What works

Auto-create project folders (Brief, WIP, Approved, FA, Audit), upload files to correct stage folder, rich card notifications to team channels, receive messages from Lark groups

Notes

Use Lark International (open.larksuite.com) if your team is outside mainland China. Use Feishu (open.feishu.cn) for China users. The API endpoints are different.

Need help? Check src/services/whatsapp.ts and src/services/lark.ts for the full implementation.
